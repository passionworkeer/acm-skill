# 计算几何经典题解

> 题目特征 → 算法选择 → 核心思路

---

## 1. 基础类型

### 点、线、向量运算
**模板**：`point.hpp`, `line.hpp`, `segment.hpp`
**核心**：
- 叉积 `cross(a,b)` → 判断方向、面积、共线
- 点积 `dot(a,b)` → 投影、垂直判定
- 两直线交点（参数式或行列式）
- 线段是否相交（快速排斥实验 + 跨立实验）

### 极角排序
**模板**：`geometry.hpp`（atan2 排序）或手写象限+叉积
**核心**：atan2 O(n log n) 简单；象限+叉积 O(n log n) 更稳定（无精度误差）

---

## 2. 凸包

### 求凸包（点集）
**模板**：`convex-hull.hpp`（Graham Scan 或 Andrew）
**核心**：按 x 排序后，分上下两条链；共线点可选保留或删除；O(n log n)

### 旋转卡壳（最远点对 / 最小面积外接矩形）
**模板**：`rotating-calipers.hpp`
**核心**：凸包两点指针同向推进，求直径、最小覆盖矩形、最远点对距离

### 动态凸包
**模板**：`dynamic-hull.hpp`
**核心**：单调栈维护上下凸壳，二分查询切线

---

## 3. 半平面交

### 求多边形交集
**模板**：`half-plane.hpp`
**核心**：有向直线极角排序 → 双端队列维护；O(n log n)；处理平行边、无解情况

---

## 4. 扫描线

### 矩形并面积
**模板**：`scanning-line.hpp`
**核心**：离散化 y 坐标，事件点（入边+1、出边-1），线段树维护覆盖长度；O(n log n)

### 矩形覆盖计数 / 矩形内点数
**模板**：`scanning-line.hpp`（改线段树查询）
**核心**：扫描线 x，维护 y 轴线段树（区间加/区间和）

---

## 5. 圆

### 圆与直线交点
**模板**：`circle.hpp`
**核心**：代数学解方程，注意判无解情况；或用向量投影法

### 圆的并面积
**模板**：`circle.hpp`（参考）或蒙特卡洛近似
**核心**：N ≤ 3 时可解析（扇形-三角形）；N 较大时用数值积分

### 最小圆覆盖（随机增量法）
**模板**：`circle.hpp`
**核心**：先 shuffle 点，依次加入；若新点在圆外，重新构造；期望 O(n)

---

## 6. 三维几何

### 三维点、向量、叉积
**模板**：`geometry-3d.hpp`
**核心**：`cross(a,b)` = 法向量；四面体体积 = `dot(a, cross(b,c)) / 6`

### 三维凸包
**模板**：`geometry-3d.hpp`
**核心**：增量法建面，三角形面定向；O(n²)

---

## 7. 多边形

### 面积（任意多边形）
**模板**：`polygon.hpp`
**核心**：叉积求和 `Σ cross(p[i], p[i+1])`，结果 /2 取绝对值

### 点在多边形内（射线法）
**模板**：`polygon.hpp`
**核心**：射线与边相交次数；注意特判共线、顶点在边上

### 凸多边形切去一半平面
**模板**：`polygon.hpp` + 半平面交
**核心**：用半平面交切凸多边形，O(n log n)

---

## 8. 常用套路

| 场景 | 套路 |
|------|------|
| 平面最远点对 | 凸包 + 旋转卡壳 |
| 点集直径 | 凸包 + 旋转卡壳 |
| 矩形并面积 | 扫描线 + 线段树 |
| 直线与凸多边形交 | 凸包 + 二分切线 |
| 判定三点共线 | 叉积 = 0 |
| 判定点在三角形内 | 面积和 = 总面积（避免精度）|
| 两圆公切线 | 化为圆心距问题，分类讨论 |
