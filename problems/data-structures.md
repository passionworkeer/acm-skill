# 数据结构经典题解

> 题目特征 → 算法选择 → 核心思路

---

## 1. 树状数组 / 线段树

### 静态区间求和 + 单点更新
**题目特征**：多次查询 `sum(l,r)`，偶尔更新某点值
**算法**：Fenwick / BIT
**模板**：`fenwick.hpp`
**核心**：`add(idx, delta)` → `sum(idx)` = 前缀和，`sum(l,r) = sum(r)-sum(l-1)`

### 区间加 + 区间求和
**题目特征**：区间增值 + 区间查询总和
**算法**：Fenwick Range Update（差分 BIT）
**模板**：`fenwick-range.hpp`
**核心**：两个 BIT，`add(B1,delta)`, `add(B2,delta*(i-1))`，前缀和 = `B1.sum * i - B2.sum`

### 区间赋值 + 区间求和（懒标记）
**题目特征**：区间操作是区间加/覆盖，查询是区间总和/最值
**算法**：Lazy Segtree
**模板**：`segtree-lazy.hpp`
**核心**：`pushDown()` 传递懒标记，合并时用 `pull()` 聚合子树信息

### 可持久化线段树（历史版本）
**题目特征**：操作后保留旧版本，支持查询历史版本
**算法**：Persistent Segtree
**模板**：`segtree-persistent.hpp`
**核心**：每次更新只改变 O(log N) 个节点，新版本共享未修改的节点

### 二维区间查询
**题目特征**：矩阵查询 / 二维前缀和
**算法**：2D BIT 或 2D Segtree
**模板**：`sparse-table-2d.hpp`, `2d-diff-array.hpp`
**核心**：2D BIT `add(x,y,delta)` + `sum(x,y)` → 前缀和；2D 差分处理矩形修改

## 2. 并查集

### 集合合并 + 连通性判断
**题目特征**：多次 union(a,b)，查询 isConnected(a,b)
**算法**：Union-Find（带路径压缩 + 按秩合并）
**模板**：`union-find.hpp`
**核心**：`find(x)` 递归 + 路径压缩；`union(a,b)` 按 size 合并

### 可持久化并查集
**题目特征**：可回退的并查集（离线分治/回滚）
**模板**：`union-find.hpp` + 手动 push/pop 栈

---

## 3. Treap

### 插入/删除/查排名/第k小（平衡树）
**题目特征**：需要有序集合，支持插入删除、第k小、排名
**模板**：`treap.hpp`（无旋转分裂合并版）
**核心**：`split(root, key)` → `(L,R)`，`merge(L,R)`，`insert/erase` = split+merge+merge

---

## 4. 莫队算法

### 区间离线询问（可移动指针）
**题目特征**：离线询问 `[l,r]`，可 O(1) 移动左右端点时更新答案
**模板**：`mo-algorithm.hpp`
**核心**：按 `l` 分块，`sort(q, block_l, sort_r)`；指针均摊 O((n+q)*sqrt(n))

### 树上莫队
**题目特征**：树上路径/子树区间查询
**核心**：Euler Tour 把树转数组区间，然后普通莫队

---

## 5. 单调队列 / 滑动窗口

### 滑动窗口最值
**题目特征**：维护大小为 K 的窗口最大值
**模板**：`sliding-window.hpp`
**核心**：维护下标单调的双端队列，队首 = 当前最大值

### 优化 DP（单调队列优化）
**题目特征**：`dp[i] = min dp[j] + cost(j,i)`，cost 满足四边形不等式
**模板**：`data-structures/monotonic-dp-hull.hpp`（或手动单调队列）

---

## 6. Link-Cut Tree

### 动态树连通性 / 路径修改
**题目特征**：树的连边/断边，动态维护路径信息（和/最大/连通性）
**模板**：`lct.hpp`
**核心**：`makeRoot(x)` → `split(x,y)` 拉出路径 → `access` 暴露节点
**典型题**：动态树最小生成树、树上路径求和/赋值

---

## 7. KD-Tree

### K 近点对 / 动态 2D 范围和
**题目特征**：二维点集，动态插入 + 矩形查询；或找第 K 近距离
**模板**：`kd-tree.hpp`
**核心**：交替按 x/y 划分，使用优先队列维护 K 近，剪枝用估价函数
**典型题**：动态矩形加、矩形求和；K 远点对

---

## 8. 凸包 / 动态凸包

### 动态维护凸包（插入点，查询切线）
**题目特征**：在线加点，动态维护上/下凸包，支持查询前缀最值
**模板**：`dynamic-hull.hpp`（或单调 DP 凸包）
**核心**：维护单调栈，查询时二分斜率

---

## 9. 常用套路

| 场景 | 套路 |
|------|------|
| 区间操作 + 区间查询 | 差分数组 + BIT |
| 离线区间询问 | 莫队（O(n√n)）|
| 树路径操作 | LCT 或 Heavy-Light + Segtree |
| 需要有序集合 | Treap |
| 前缀最值 / RMQ | Sparse Table（O(1) 查询）|
| 矩形操作 | 扫描线 + Segtree |
